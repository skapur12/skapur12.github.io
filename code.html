Attack of the Space Aliens

Using C, I created a game for the GameBoy Advance called ‘Attack of the Space Aliens.’ The player controls a spaceship (which they can move left, right, up, and down) and must press buttons to shoot bullets at aliens (both big and small) on the sides of the screen.

The player must take all of the big aliens’ lives to win. The player will also lose a life if they are hit by an alien or they encounter a yellow star. There are 3 levels in total.

Donation Tracker

Alongside a few team members, I built an application in Android Studio using Java to help non-profit organizations keep track of donations of goods. The app allowed organizations to track and categorize and categorize donations. It tracked distribution and donation metrics, and it connected donation centers to each other to allow for redistribution of goods (if necessary).

This app followed a Model-View-Controller pattern. My teammates and I took turns working on each of the aspects each week, so I helped develop the Model, the View, and the Controller.

Personal Website

I’m including this website on the list! I created it in GitHub Pages using HTML and CSS. As you can see, it contains my resume, descriptions of projects (both code-based and design-based) that I’ve done, links to my profiles, and my blog.

I created a default layout that can be applied to every page, which simplifies my task quite a bit! Instead of duplicating the same code on every page, I can simply use this layout to place the same tabs at the top of the page and icons at the bottom of the page.

I also used Markdown (Kramdown specifically) and Jekyll to create my blog and integrate it onto this site. (More posts are coming soon!) Thanks to these handy tools, I can write my blog posts in a plain text file and hit publish, and they’ll take care of all of the formatting for me.

Lastly, I used JavaScript to add a Disqus comment section into each blog. Now, it incorporates all of the features of Disqus (tweeting/sharing posts, reacting to posts, and commenting)!
